We expect this exercise to take you somewhere between 15-minutes to 2 hours. If you take more than two hours, please post what you tried along wtih error messages to our slack channel.

In keeping with our attempt to help you think in different ways, we have listed three approaches that work. There is no shame or pride in choosing anyone of them. Just choose the method you are comfortable with and proceed.

 

Approach 1: Try it out yourself

  1. We'd like to promote independent thinking and problem solving. So if you feel ready, try to install Python without referring to any documentation, go for it!

 

Approach 2: Google for instructions

  1. Google for visual studio code install 'your operating system name'
  2. Follow the instructions

 

Approach 3: follow instructions

  1. Visit https://code.visualstudio.com/download to install visual studio code for your operating system
  2. Visit https://code.visualstudio.com/download to install python extension for vscode
  3. Start python coding on vscode
  4. Vist https://marketplace.visualstudio.com/items?itemName=ms-python.python help documentation for any questions

 

Exercise

Once you are done, write some python code or import any python project on vscode

 

Verify your setup

After you are done with the exercise listed above, it is time to verify your vscode setup. It is a good habit to take small steps and verify them immediately. This will help you avoid getting stuck later on. To verify your setup execute python code on vscode. You should see python code runs on vscode with out any errors
